About Me

Mengenal lebih dekat tentang background dan keahlian saya sebagai Android Developer

Saya adalah seorang Android Native Developer pemula yang tengah mendalami dunia pengembangan aplikasi mobile. Dengan semangat belajar yang tinggi dan ketertarikan mendalam terhadap teknologi Android, saya telah menyelesaikan sejumlah proyek individu sebagai bagian dari upaya untuk mengasah kemampuan teknis dan kreativitas saya.

Keahlian saya meliputi pengembangan aplikasi Android menggunakan Kotlin, implementasi modern architecture patterns seperti MVVM, MVI dan Clean Architecture, serta integrasi dengan berbagai API dan layanan cloud. Saya juga aktif mengikuti perkembangan teknologi Android terbaru termasuk Jetpack Compose untuk UI development.

Selain technical skills, saya memiliki kemampuan problem-solving yang kuat dan senang berkolaborasi dalam tim untuk menciptakan solusi teknologi yang berdampak positif.
